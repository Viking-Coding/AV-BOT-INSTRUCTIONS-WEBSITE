Event Creation

Command: /create_event
Subcommands: None

Command Permission: ADMIN
Result Usage: All Users

Every aspect is editable by you; Create events that don't require roles, events that give a role in the Racing server, and events that give roles in the main server.
Admins, Run the command, edit the fields then click Send
After you have sent the Event to a channel, Right click -> Apps -> Roles to assign a role for those signed up with Yes

Fuel Calculator

Command: /fuel
Subcommands: User Information

Command Permission: All Users
Result Usage: Ephemeral

Calculates fuel required with a 1 lap buffer; Also specifies how many pitstops will be required, along with how much fuel to refill in the final stop.

Ticket System

Command: /ticketing_init
Subcommands: None

Command Permission: ADMIN
Result Usage: All Users

Run the command once in the channel of your choice to generate the announcement and buttons.
Then users can click the buttons and fill in the prompts. Once they press submit it will create a private thread for them and the required admin role to discuss the ticket.

Google Calendar

Command: /google_calendar
Subcommands: Simracing, IRL Motorsport

Command Permission: All Users
Result Usage: All Users

Automatically posts on Sunday 8pm the next weeks Simracing and IRL Motorsports AV calendars to the channel of your choice.
There is a manual command available as well.
